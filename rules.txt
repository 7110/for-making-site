# cording rules

## 0. Editor
- ATOM を使用
- ATOM(menu bar) → Preference → install
	→ editorconfig を検索してインストール


## 1. HTML
- インデントは半角スペース4個とする。
- 入れ子になったHTMLは必ず1段階だけインデント（半角スペース4個）する。
- シングルクォートは使わず、ダブルクォートを使う。
- HTML5を使用する。
- lang要素を設定する。
- 「IE互換モード」の設定をする。
- 文字コードはUTF-8で設定する。
- JavaScriptの読み込みはbodyタグの閉じタグの前に行う。


## 2. CSS（Sass）
### 基本事項
- インデントは半角スペース4個とする。
- 可読性のために、開き波括弧の前に1つの半角スペースを入れる。
- 閉じ波括弧は新しい行に記述する。
- 「:」のあとに1つの半角スペースを入れる。
- idは使用せずclassを使用すること
- インラインCSSは使わない。
- 文字コードはUTF-8で設定する。
- 色の指定はHEX型を使用し、すべて小文字にする。
- 可能な場合は省略形のHEX型の値を使う。
- Sassのネストは3階層まで。

### セレクタの書き方
- Blockの最初の文字は大文字で書く。
- Elementの最初の文字は小文字で書く。
- BlockとElementは「__」でつなぐ。
- Modifierをつける場合は「--」でつなぐ。
- レイアウトに関するものは「l-」の接頭辞をつける。
- JavaScriptで状態を変更する場合は「isXX」という命名にする。
